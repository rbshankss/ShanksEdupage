# Shanks Education Website

A React web application based on the Shanks Education promotional poster.

## Features

- Modern, responsive design matching the original poster
- Dark theme with red and white accent colors
- Contact form popup with fields for:
  - Name
  - Phone Number
  - Email
  - Year Level
  - Subjects of Interest

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will start on `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- CSS3
